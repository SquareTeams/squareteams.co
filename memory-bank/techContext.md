# Tech Context

## Technologies Used
- **Framework**: Next.js 14.0.4
- **Language**: TypeScript 5
- **UI Library**: React 18
- **Styling**: Tailwind CSS 3.3.0
- **Animation**: GSAP, Anime.js, Locomotive Scroll (to be consolidated to GSAP)
- **Linting**: ESLint

## Technical Constraints
- The project must be deployable on Vercel.
- The application should be performant and optimized for search engines.
